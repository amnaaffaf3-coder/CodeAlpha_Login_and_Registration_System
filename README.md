# CodeAlpha_Login_and_Registration_System

## Description
A beginner-friendly C++ console application that allows users to register and log in using a username and password. The project demonstrates file handling, conditional statements, loops, functions, and user input validation.

## Features
- User Registration
- User Login
- Stores user credentials in a file
- Input validation
- Simple menu-driven interface

## Technologies Used
- C++
- File Handling
- Functions
- Loops
- Conditional Statements
